## Ejecutar en Desarrollo

1. Clonar Repositorio
2. Ejecutar
```bash
$ npm install
```
3. Tener NestJS CLI Instalado
```bash
$ npm i -g nestjs/cli
```
4. Levantar Base de Datos
```bash
$ docker-compose up -d
```
5. Ejecutar para levantar
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

6.  Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
